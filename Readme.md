# 🚗 DriveXplorer

## 📌 Project Overview

DriveXplorer is a web application that fetches and displays real-time vehicle manufacturer data using the NHTSA API. It allows users to interact with the data through search, filtering, sorting, and UI enhancements like dark mode.

---

## 🎯 Features

### 🔍 Search

* Users can search car brands in real-time.
* Implemented using JavaScript `filter()`.

### 🎯 Filtering

* Filter results based on criteria (e.g., ID > 1000).
* Uses `filter()` higher-order function.

### 🔽 Sorting

* Sort data alphabetically (A–Z / Z–A).
* Implemented using `sort()`.

### ❤️ Button Interaction

* Like button added to each card.
* Displays interactive feedback.

### 🌙 Dark Mode

* Toggle between light and dark themes.
* Enhances user experience.
